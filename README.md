# Система контроля посещаемости завода коньяка

Простая система для отслеживания прихода и ухода работников завода.

## Функциональность

- **Две роли пользователей:**
  - Администратор: полный доступ к системе, просмотр всех отчетов
  - Работник: только отметка прихода/ухода

- **Основные возможности:**
  - Отметка прихода на работу
  - Отметка ухода с работы
  - Просмотр текущих сотрудников на работе
  - Отчеты по посещаемости (только для админов)
  - Детальная информация о каждом работнике

## Структура проекта

Проект разделен на части между тремя разработчиками:

- **Максим**: Модели данных (`attendance/models.py`)
- **Миша**: Аутентификация и роли (`attendance/admin.py`, `attendance/forms.py`)
- **Артур**: Логика посещаемости и интерфейс (`attendance/views.py`, `attendance/urls.py`, шаблоны)

## Запуск проекта

1. Установите зависимости:
```bash
pip install django
```

2. Создайте миграции и примените их:
```bash
python3 manage.py makemigrations attendance
python3 manage.py migrate
```

3. Создайте тестовые данные:
```bash
python3 init_data.py
```

4. Запустите сервер:
```bash
python3 manage.py runserver
```

5. Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/
